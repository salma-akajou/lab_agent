# Rule : Convention de Nommage

Cette règle impose une convention de nommage uniforme
pour l’ensemble du projet.

## Objectif :
- Rendre le code plus clair et plus professionnel.
- Éviter les ambiguïtés dans les noms des fichiers et des classes.
- Faciliter la lecture et la collaboration en équipe.

## Conventions à respecter :
- Classes : PascalCase (ex : FilmService)
- Méthodes : camelCase (ex : createFilm)
- Variables : camelCase
- Fichiers Blade : kebab-case (ex : edit-film.blade.php)

## Résultat attendu :
Un code cohérent, facile à lire et simple à maintenir.
